# Алгоритмические эксперименты и количественный анализ

В репозитории собраны проекты, посвящённые исследованию алгоритмических методов аппроксимации, регуляризации и сжатия данных.

## Подход

В рамках проектов я:

- сравнивала альтернативные алгоритмические подходы;
- проектировала серию вычислительных экспериментов;
- выбирала и использовала количественные метрики качества (Max Error, MSE, PSNR);
- анализировала влияние параметров на стабильность и точность результатов;
- формулировала выводы на основе полученных данных и сравнительного анализа.

## Фокус

Экспериментальный дизайн, работа с метриками и принятие решений на основе количественных результатов.
