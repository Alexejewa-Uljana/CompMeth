# Решение интегральных уравнений Фредгольма I рода методом регуляризации

## Тестовые примеры

### Пример 1: Хорошо обусловленное уравнение
```
∫[0,1] (1 + xt) u(t) dt = 1 + x/2
```
**Точное решение:** `u(x) = 1`

### Пример 2: Плохо обусловленное уравнение
```
∫[0,1] cos(π(x-t)) u(t) dt = (2/π)·sin(πx)
```
**Точное решение:** `u(x) = sin(2πx)`

## Ключевые наблюдения

### Сходимость метода

- **Хорошо обусловленные задачи**: ошибка монотонно уменьшается с α → 0
- **Плохо обусловленные задачи**: существует оптимальный α (не слишком малый!)
- **Лучшая точность для хорошо обусловленных**: α = 0.001 (ошибка = 0.003293)
- **Лучшая точность для плохо обусловленных**: α = 5.0 (ошибка = 1.081719)

### Эффекты некорректности

- **При очень малых α**: неустойчивость в плохо обусловленных задачах
- **При больших α**: слишком сильная регуляризация, большие ошибки
- **Оптимальные зоны**:
  - Хорошо обусловленные: α ∈ [0.01, 0.1]
  - Плохо обусловленные: α ∈ [0.05, 0.2]
