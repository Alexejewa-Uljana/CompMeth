# Метод сплайн-коллокаций для уравнений Фредгольма 2 рода

## Описание задачи

Реализация и тестирование модифицированного метода сплайн-коллокаций для решения интегральных уравнений Фредгольма II рода вида:

```
u(t) = f(t) + ∫[a,b] K(t,x)u(x) dx
```

## Реализованные тесты

### Тест 1: Экспоненциальное ядро
**Уравнение:**
```
u(t) - 0.5∫[0,1] e^(t-x) u(x) dx = 0.5e^t
```
**Точное решение:** `u(t) = e^t`

**Результаты:**
- h = 0.2: ошибка = 0.00249
- h = 0.1: ошибка = 0.00083  
- h = 0.05: ошибка = 0.00067

### Тест 2: Полиномиальное ядро
**Уравнение:**
```
u(t) - ∫[0,1] t·x u(x) dx = (2/3)t
```
**Точное решение:** `u(t) = t`

**Результаты:**
- h = 0.2: ошибка = 0.00413
- h = 0.1: ошибка = 0.00086
- h = 0.05: ошибка = 0.00033

## Ключевые наблюдения

### Сходимость метода
- **Монотонное улучшение точности** с уменьшением шага сетки
- **Экспоненциальное ядро:** ошибка уменьшилась в 3.7 раза (0.00249 → 0.00067)
- **Полиномиальное ядро:** ошибка уменьшилась в 12.5 раза (0.00413 → 0.00033)

### Точность решения
- **Высокая точность** достигнута даже при относительно больших шагах
- **Лучшая сходимость** для полиномиального ядра
- **Стабильность** метода подтверждена на различных типах ядер

## Особенности реализации

### Используемые методы
- **B-сплайны** в качестве базисных функций
- **Трехточечные аппроксимационные функционалы** для вычисления коэффициентов
- **Локальная аппроксимация** с использованием методов квазиинтерполяции

### Параметры вычислений
- **Шаги сетки:** 0.2, 0.1, 0.05
- **Количество точек интегрирования:** 20
- **Диапазон:** [0.0, 1.0]
