# Распараллеливание схем аппроксимации B-сплайнами

## Стратегия распараллеливания

### 1. Распараллеливание схемы уменьшения вариации

**Параллелизуемая операция**: вычисление контрольных точек

**Способ реализации**: 
- Каждая контрольная точка вычисляется независимо по формуле cⱼ = f(tⱼ*)
- Использование многопоточности для параллельного выполнения циклов
- Разделение диапазона индексов j = 1,...,n между потоками

**Ожидаемое ускорение**: линейное относительно числа ядер процессора

### 2. Распараллеливание трехточечного функционала

**Параллелизуемая операция**: заполнение матрицы системы уравнений

**Способ реализации**:
- Независимое вычисление каждой строки матрицы
- Параллельная обработка точек коллокации
- Разделение работы по строкам матрицы между потоками

**Особенности**:
- Каждая строка содержит только 3 ненулевых элемента
- Минимальные конфликты при доступе к памяти
- Правая часть системы вычисляется параллельно с матрицей
